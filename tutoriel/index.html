Un Sudoku est un jeu en forme d'une grille, plusieurs variations de Sudoku existent, mais le plus connu est celui de 9x9.
Dans ce tutoriel, nous allons utiliser une grille de 4x4.
Le but du Sudoku est de remplir tous les cases en suivant deux règles.
Ces règles sont:
1. Chaque ligne et colonne ne peut pas avoir le même chiffre
et
2. Chaque petite grille ne peut pas avoir le même chiffre
Dans une grille de 4x4,
vous pouvez seulement utiliser les chiffres 1 à 4.
Pour remplir une case, vous devez premièrement selectionner une case en le cliquant dessus.
Ensuite, avec votre clavier ou les boutons, vous pouvez insérer un chiffre dans la case selectionnée.
Si vous avez faites une erreur, vous pouvez toujours vider une case selectionnée en appuyant sur la touche de retour arrière.
Lorsque vous avez fini le Sudoku, vous pouvez appuyer
sur la touche Entrée ou cliquer le bouton Vérifier.
Dans les Sudokus de 9x9 et de 16x16, c'est le même concept,
la seule nuance est qu'il y a plus de cases a remplir avec plus de chiffres.
Pour commencer à jouer au Sudoku, choisi entre les difficultés: Facile (4x4), Normal (9x9) et Difficile (16x16) dans la barre de navigation et amusez-vous!
